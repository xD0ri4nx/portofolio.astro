A high-performance, static-first developer portfolio featuring a custom Synthwave/Glassmorphism design. Built to achieve a perfect 100/100 Core Web Vitals score by shipping zero JavaScript to the client by default.

🌐 **Live Demo:** [xD0ri4nx.github.io/portfolio](https://xd0ri4nx.github.io/portofolio.astro/)

---

## ⚡ Features

* **Static-First Architecture:** Built with Astro v6 for lightning-fast page loads.
* **Modern Styling engine:** Utilizes the new CSS-first Tailwind v4 with `@tailwindcss/vite`.
* **Synthwave Aesthetic:** Custom UI featuring deep purples, neon pinks, glassmorphism overlays, and dynamic hover states.
* **Fully Responsive:** A dynamic bento-grid layout that adapts perfectly from mobile devices to ultrawide monitors.
* **Automated CI/CD:** Fully integrated with GitHub Actions for automated building and deployment to GitHub Pages on every push to the `main` branch.

## 🛠️ Tech Stack

* **Framework:** Astro 6.x
* **Styling:** Tailwind CSS 4.x
* **Tooling:** Vite, Node.js
* **Deployment:** GitHub Pages & GitHub Actions workflows
* **Language:** TypeScript / HTML / CSS
